# S.A. Construction Front End

This is the public website for S.A. Construction and can be found at [www.saconst.com](https://saconst.com).

## Development

To run this project locally:

```
  npm run dev
```

## Project Dependencies
Any backend request will not be connected. To work with the backend, you can [download the SA Construction Backend Repo Here](https://github.com/pblake313/SA-Construction-API)


## Build Project

To create a production version:

``` 
  npm run build
```
